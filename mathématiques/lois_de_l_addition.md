# Lois de l'addition

Il y a de multiples lois qui composent l'addition. En voici cinq.

Associative : a + (b + c) = (a + b) + c
Commutative : a + b = b + a
Distributive : a(b + c) = ab + ac
Contient un élément d'identité : a + 0 = 0 + a = a
À un inverse pour chaque élément : a + (-a) = (-a) + a = 0
